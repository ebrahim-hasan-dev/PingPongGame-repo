# Ping Pong Game

A lightweight arcade-style game built with Python. This project is designed to test reflexes and provides a clean, documented example of basic game logic and GUI implementation.

## Features
* **Competitive playing style:** The ball moves towards one of the bats, with the ball's speed increasing with each stroke..
* **Score System:** Make the ball hit the opponent's wall.
* **Clean Architecture:** Organized Python source files for easy reading and maintenance.
* **Portable:** Can be converted to a standalone executable for Windows.

## Tech Stack
* **Language:** Python 3.x
* **IDE:** Visual Studio Code
* **Deployment:** PyInstaller (for .exe creation)

## Installation & Setup
To run this project locally, ensure you have Python installed, then follow these steps:

1. Clone the repository:
   git clone [https://github.com/ebrahim-hasan-dev/PingPongGame-repo.git]

2. Navigate to the project directory:
   cd PingPongGame-repo

3. Run the main file:
   python main.py

## How to Play
* **Move:** Use the w and s keys for player one and use the up and down Arrow keys for player two on your keyboard.
* **Goal:** Make the ball go away from your opponent's racket as possible to increase your score.
* **Game Over:** The ball hitting the opponent's wall or your wall.
